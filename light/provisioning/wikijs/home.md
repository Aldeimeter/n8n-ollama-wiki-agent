# Welcome

This Wiki.js instance was provisioned automatically as part of the
Vagrant stack (PostgreSQL + Ollama + n8n + Wiki.js).

## Stack

- **PostgreSQL** — shared database (`192.168.56.13`)
- **Ollama** — local LLM API on port `11434` (`192.168.56.12`)
- **n8n** — workflow automation on port `5678` (`192.168.56.11`)
- **Wiki.js** — this wiki on port `3000` (`192.168.56.14`)

## Getting started

Edit this page from the toolbar, or create new pages from the **New Page**
button. This page lives at the `home` path and is served at the site root.
